# Documentation Rules

## Structure

- Docs are numbered 01-12 in `docs/`
- Each doc starts with `# <number>. <Title>`
- Cross-reference other docs with relative links: `[text](XX-filename.md)`
- Index page is `docs/README.md`

## Style

- Use numbered sub-headings: `## 3.1 Section Title`
- Include code blocks with language tags for all commands and code snippets
- Use `> **Note.**` / `> **Warning.**` callout blocks for important asides
- Keep explanations practical — explain *why*, not just *how*
- End each doc with a link back to `docs/README.md`
